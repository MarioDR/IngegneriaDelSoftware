◆ clickAggiungi()

void gruppo15.ingegneriadelsoftware.controller.ScenaAggiungiLibroController.clickAggiungi ( ActionEvent event)
private

Il metodo è collegato all'azione del pulsante 'aggiungiButton' che permette di aggiungere un libro al catalogo con tutte le operazioni di inserimento nel file CSV e nella collezione di gestione. La gestione della sintassi corretta è rimandata all'interfaccia grafica.

Precondizione
il nuovo libro deve essere legittimo (questo metodo non ha questa responsabilità)
Postcondizione
il nuovo libro verrà aggiunto al GestoreLibri
tutti i dati del nuovo libro vengono custoditi nel file CSV corrispondente
Parametri
eventl'evento del click sul pulsante
