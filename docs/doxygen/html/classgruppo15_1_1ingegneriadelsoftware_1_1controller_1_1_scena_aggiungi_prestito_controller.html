◆ clickAggiungi()

void gruppo15.ingegneriadelsoftware.controller.ScenaAggiungiPrestitoController.clickAggiungi ( ActionEvent event)
private

Il metodo è collegato all'azione del pulsante 'aggiungiButton' che permette di aggiungere un prestito alla lista con tutte le operazioni di inserimento nel file CSV e nella collezione di gestione. La gestione della sintassi corretta è rimandata all'interfaccia grafica.

Precondizione
il nuovo prestito deve essere legittimo (questo metodo non ha questa responsabilità)
Postcondizione
il nuovo prestito verrà aggiunto al GestorePrestiti
tutti i dati del nuovo prestito vengono custoditi nel file CSV corrispondente
Parametri
eventl'evento del click sul pulsante
